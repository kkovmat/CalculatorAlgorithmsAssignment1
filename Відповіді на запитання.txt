1. Структура даних - це те, в який спосіб дані збережені в комп'ютері. Інтерфейс структури даних - це правила роботи з цією структурою даних.
2. RAM - це модель обчислення для аналізу ефективності алгоритму. Big-O використовується для визначення часової та просторової складності алгоритму.
3. Це означає, що середній час виконання алгоритму дорівнює n^2, де n - кількість вхідних даних. Зазвичай n^2 означає два цикла, один з яких вкладений в інший.
4. Стек - це структура даних, яка зберігає дані за принципом Last In First Out (LIFO), тобто елемент кладемо на вершину стеку і дістаємо з вершини. Стек можна реалізувати, наприклад, за допомогою array list. У черзі принцип - First In First Out.
5. FIFO - First In First Out, працює як реальна черга. LIFO - Last In Last Out, працює як стопка паперів.
